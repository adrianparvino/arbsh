Arbitrary Precision Mathematics Library
BUILDING:

	cd arbprec
	make 



TESTING:

	Run the test binary:

		./tests/test-arbitrary-precision number number base scale

	Run the algorithm D division test suite:

		make test



USING THE API:

	Look inside of the various tests inside of tests/ for examples on how
	to use the API.
