Arbitrary Precision Mathematics Library





BUILDING:

	cd arbprec
	make 




TESTING:

	Run the test binary:

		./tests/arbprec-driver 988983483.14891487417128 908408.38034834834857345834




API:

	Example API usage for dividing two numbers:

	int main(void)
	{
		bigflt *flt1 = str_to_bigflt("9828954325778328944734987.2398472984356790934");
		bigflt *flt2 = str_to_bigflt("78289764322362423523525287.327320000038908953476");

		bigflt *flt3 = arbprec_expand_vector(NULL, flt1->len + flt2->len); 

		flt1 = arbprec_add_precision(flt1, scale);
	        arbprec_match_precision(flt1, flt2);


		flt3 = arbprec_div(flt1, flt2, flt3);

		arbprec_print(flt3);
		
		return 0;
	}


	See tests/arbprec-driver.c for another example on how to use the API.
