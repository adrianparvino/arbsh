Arbitrary Precision Math Library by Graff and POSIX `bc' by Gavin Howard

	An arbitrary precision implementation, readline implementation and
	POSIX bc implementation written by Graff, Howard, Hexing et al.
BUILDING:

	cd arbprec
	make 



TESTING:

	Run the test wrappers and binaries:

		./tests/divregression.sh
		./tests/mulregression.sh
		./tests/subregression.sh
		./tests/addregressions.sh

	Run division with an increased scale size:

		./tests/algd 123123.23 12.123 10 10000

	Other binaries:

		./tests/add  123123612837612836871212.123 10 10
		./tests/sub  123123612837612836871212.123 10 10
		./tests/mul  123123612837612836871212.123 10 10
		./tests/nsqrt 123123612837612836871212.123 10 10



USING THE API:

	Look inside of the various tests inside of tests/ for examples on how
	to use the API.
