
Basic libsdl development for debian
-----------------------------------

  apt-get update

  apt-get install libsdl-image1.2 libsdl-image1.2-dev libsdl-mixer1.2 libsdl-mixer1.2-dev

  tar -xf /usr/share/doc/libsdl1.2-dev/examples/examples.tar.gz

  cd test

  gcc testalpha.c -L/usr/lib/x86_64-linux-gnu  -I/usr/include/SDL/  


If all went as expected a window should appear.
